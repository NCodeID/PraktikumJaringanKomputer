### Cara menggunakan
1. **Jalankan Server**
   - UDP:  
     ```bash
     python UDPServer.py
     ```
   - TCP:  
     ```bash
     python TCPServer.py
     ```

2. **Jalankan Client**
   - UDP:  
     ```bash
     python UDPClient.py
     ```
   - TCP:  
     ```bash
     python TCPClient.py
     ```
